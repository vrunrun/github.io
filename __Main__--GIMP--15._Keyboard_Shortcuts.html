Keyboard Shortcuts



• Studies had shown that if you use keyborad shortcuts, you can save 1 hour for every 8 hour work

# Configure keyboard shortcuts
Edit > Keyboard Shortcuts


General



KeyUsage
ctrl-Mouse WheelZoom in and out (from where the mouse is pointed to)
Space_bar-Mouse_MoveMove image
ctrl-zUndo
ctrl-yRedo
ctrl-aSelect all (marching ants)
ctrl-shift-aSelect none
/Access Search function for features, opened files
ctrl-fRepeat filter or image adjustment
ctrl-sSave composition as .XCF file
ctrl-shift-sSave composition as a new .XCF file
ctrl-shift-eExport composition (.jpg, .png, pdf, etc.)

Foreground and Background Color



KeyUsage
dReset foreground & background colors to b&w
xSwap foreground & background colors
shift-qToggle Quick Mask on & off


• When using Quick Mask using the b&w foreground & background, if you paint too much, you can revert the change with the “x” key

~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~

Selection Tools



• These are equivalent to clicking the icon Mode under Tool Options
• The following keys are usable only after a selection is made with Selection Tools

KeyUsage
shiftAdd to current selection
ctrlSubtract from current selection
ctrl-shiftIntersect current selection


~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~

Paint Tools



KeyUsage
shiftDraw straight line
shift-ctrlStraight line mode with 15 degree increment
ctrl-clickActivates Eyedropper Tool to pick a color Pick foreground color for Brush, Pencil etc. Pick background color for Eraser
