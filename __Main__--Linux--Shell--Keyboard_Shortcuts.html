Keyboard shortcuts


• F11: maximize window
• Bash default shortcuts
◇ Tab + Tab: show all possible options
◇ Ctrl+L: clear screen (same as clear)
◇ Ctrl+A: go to the beginning of the line
◇ Ctrl+E: go the the end of the line
◇ Ctrl+R: reverse search (repeat to next hit)
◇ Ctrl+D: close window
◇ Ctrl+G: back to cursor
