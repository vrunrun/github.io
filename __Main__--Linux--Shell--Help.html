Online Cheatsheet



curl cheat.sh/[command]


~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~

Get Help on command



cd --help
# or
help cd
info ls
whatis ls                   # one line summary of a command


~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~

If you don't remember the exact command, try:



# Find command with keyword
man -k owner
# or
apropos password


~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~

Check if the command is available



# Locate command
which ls
which cd        # no output because cd is a buildin shell command


~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~

Check Command Types



# Show command types: shell command, executable, link
type cd
type python
type ll
