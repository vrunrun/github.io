How to Download and Install GIMP

1:00

#Install Flatpak
sudo apt install flatpak # Debian 10 buster and newer
# or 
sudo apt -t stretch backports flatpak # Debian 9

# Install Flatpak plugin
sudo apt install gnome-software-plugin-flatpak

# Add the Flathub Repository
sudo  flatpak remote-add --if-not-exists flathub https://flathub.org/repo/flathub.flatpakrepo

# Restart the system

# Install GIMP
sudo flatpak install https://flathub.org/repo/appstream/org.gimp.GIMP.flatpakref


Customizing the GIMP Theme/User Interface

3:02

# Customize theme and UI
Edit > Preferences > Interface > Theme
Edit > Preferences > Interface > Icon Theme
Edit > Preferences > Default Image


• Set image Size (for File > New): width=1920, height=1080
• X/Y resolutions (PPI for final image): 300 (for print), 72 (for digital display)

Image Window Overview

18:03

• You can see the file name and size at the bottom of the Image Window
• Ruler display on top and left borders
• Can change unit of display like px (default), inches @ bottom border
• Can also change zoom level @ bottom border
• For large image, use the icon “Navigate Image Display” @ bottom right corner of the Image Window (alias “Bird Eye View” in Photoshop)
• To maximize display image, select icon “Zoom Image When Window Size Change” at the top right corner of the Image Window
• You can access the Mini Menu icon at the top left corner of the Image Window
• Use Quick Mask Tool icon @ bottom left corner (for advance use)
