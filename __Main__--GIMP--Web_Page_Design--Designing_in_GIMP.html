Alignment



• Turn on the Grid system (ctrl-;)



• Align logo to the left. Keep the logo contained within the border
• The blue dotted is called a gutter / padding
• To align logo top to bottom,

Method #1: Use Selection Tool

/* Align Top to Bottom */
1. Use "Rectangle Select Tool" to draw top to bottom boundary
2. Use "Align Tool" and select the appropriate tool options
- "Relative to: Selection"
3. Make sure that the top layer Grid System is turned off
3. Click on the Logo to select


• The square boxes around the logo indicate the logo is selected
